# 8. MVP 기능 우선순위

작성일: 2026-06-14

## 질문

- 좋아요/싫어요 피드백은 MVP에 넣을 것인가, 후순위로 둘 것인가?

## 사용자 답변

- 좋아요/싫어요 피드백은 결국 넣고 싶다.
- 다만 1차 개발에는 안 넣어도 될 것 같다.

## 정리된 결정

좋아요/싫어요 피드백은 장기적으로 추천 품질 개선에 중요하지만, 1차 개발에서는 제외한다.

첫 MVP는 성향 기반 추천과 대화형 바이브 수정 경험 검증에 집중한다.

## 필수

- 회원가입
- 로그인
- 가입 시 기본 데이트 성향 문장 선택
- 현재 지도 화면 영역 기반 플레이스 조회
- 저장된 성향 기반 플레이스 추천
- 지도 기반 플레이스 표시
- 플레이스 리스트 표시
- 추천 점수 정렬
- 플레이스 간단 요약
- 추천 이유
- 외부 링크
- 채팅창 입력
- 자연어 기반 추천 수정
- 자연어 기반 코스 생성

## 중요

- 내 성향 수정
- 오늘 데이트 조건 추가 선택
- 추천 점수 기반 마커 강약 표시
- 평점 표시
- 장소 추가
- 장소 삭제
- 장소 교체
- 코스 순서 변경
- 코스 구조 변경
- 분위기 변경
- 이동 거리 기준 수정
- 모호한 요청에 대한 추가 질문
- 추천 결과/코스 재생성

## 후순위

- 좋아요/싫어요 피드백
- 코스 저장
- 이전 추천 기록
- 고급 성향 분석
- 관리자 검수 화면 고도화
- 장소 등록 요청
- 공유 기능
