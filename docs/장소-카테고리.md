# Vibe Dating Place Categories

작성일: 2026-06-14

이 문서는 Vibe Dating에서 직접 정의해 관리하는 데이트 장소 카테고리의 초기 기준이다.

외부 지도 서비스의 카테고리를 그대로 사용하지 않고, 데이트 맥락에 맞게 대분류와 중분류를 나누어 관리한다.

## 카테고리 체계

| 대분류 | 중분류 |
| --- | --- |
| 식사 | 한식, 양식, 중식, 일식, 아시안, 고기, 해산물, 분식, 뷔페, 비건 |
| 카페·디저트 | 일반 카페, 베이커리, 디저트, 전통찻집, 루프탑, 뷰카페, 북카페, 테마카페 |
| 문화·관람 | 영화관, 미술관, 박물관, 전시, 공연, 연극·뮤지컬, 서점, 아쿠아리움, 동물원·식물원 |
| 체험·액티비티 | 방탈출, 보드게임, 오락실·VR, 볼링, 탁구, 당구, 클라이밍, 공방, 원데이클래스 |
| 산책·자연·야외 | 공원, 산책로, 강변·호수, 해변, 숲·수목원, 전망대, 야경명소, 드라이브, 피크닉 |
| 쇼핑·구경 | 백화점, 쇼핑몰, 아울렛, 편집숍, 소품숍, 빈티지숍, 서점, 시장, 복합문화공간 |
| 술·야간 | 와인바, 칵테일바, 펍, 이자카야, 전통주점, 루프탑바, 재즈·라이브바, 야시장 |
| 휴식·힐링 | 스파, 마사지, 찜질방, 온천, 호텔 라운지, 휴양림, 호캉스 |

## 운영 원칙

- 카테고리는 Vibe Dating의 추천과 바이브 데이트 코스 수정에 사용하는 내부 기준이다.
- 외부 API의 업종명이나 카테고리명은 이 기준으로 매핑해 사용한다.
- 하나의 장소가 여러 맥락에 걸칠 수 있지만, MVP에서는 대표 카테고리 1개를 우선 저장한다.
