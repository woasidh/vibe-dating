# 2. 타깃 사용자와 사용 상황

작성일: 2026-06-14

## 질문

- 이 서비스를 가장 먼저 써볼 사람은 어떤 커플인가?
- 데이트 장소를 고를 때 보통 누가 주도하는가?
- MVP에서 처음 집중할 지역은 어디인가?
- 어떤 유형의 데이트 장소를 먼저 잡고 싶은가?

## 사용자 답변

- 어떤 커플이든 데이트코스를 짜는 사람들이다.
- 보통 한 사람이 자주 찾는다.
- 우선은 유명한 곳 위주로 특정 지역만 오픈하고, 서비스가 흥행하면 추후 다른 지역도 오픈할 예정이다.
- 데이트가 가능한 모든 장소는 포함시키려고 한다.

## 정리된 결정

Vibe Dating의 초기 타깃은 특정 연령, 연애 기간, 직업군으로 제한하지 않는다.

핵심 타깃은 데이트코스를 짜야 하는 모든 커플, 특히 데이트 전 장소를 직접 찾아보고 비교하는 사람들이다.

데이트 장소 탐색은 커플 두 사람이 똑같이 나눠서 하기보다, 한 사람이 더 자주 맡는 경우가 많다. 따라서 MVP는 둘이 같이 쓰는 앱이기 이전에, 데이트 계획을 주도하는 한 사람이 빠르게 좋은 후보를 찾는 도구로 출발한다.

초기에는 유명 데이트 상권 또는 핫플 지역 일부만 오픈한다. 서비스 반응이 좋으면 이후 다른 지역으로 확장한다.

장소 유형은 음식점, 카페, 술집, 전시, 팝업, 공방, 산책 장소, 쇼핑, 액티비티 등 데이트 가능한 모든 장소를 포함 대상으로 본다.
