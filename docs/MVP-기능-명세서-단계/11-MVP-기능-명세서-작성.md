# 11. MVP 기능 명세서 작성

작성일: 2026-06-14

## 진행 내용

앞 단계에서 결정한 내용을 바탕으로 MVP 기능 명세서 1차본을 작성했다.

작성된 문서:

- [../MVP-기능-명세서.md](../MVP-기능-명세서.md)

## 문서에 포함된 주요 섹션

- 문제정의
- 타깃 사용자와 사용 상황
- 핵심 가치와 제품 가설
- MVP 목표와 제외 범위
- 핵심 사용자 플로우
- 도메인 모델
- 기능 후보 목록
- MVP 기능 우선순위
- 화면 단위 요구사항
- 데이터 및 정책 요구사항
- MVP 기능 명세
- 검증 기준

## 기능 명세에 포함된 주요 기능

- 회원가입
- 로그인
- 성향 선택 온보딩
- 지도 기반 플레이스 추천
- 플레이스 카드
- 바이브 수정 채팅
- 코스 생성
- 코스 수정

## 문서 작성 원칙

- 사용자의 답변을 기준으로 결정 사항을 정리한다.
- 아직 기술적으로 확정되지 않은 추천 로직은 별도 기술 검토 문서로 분리한다.
- MVP 기능 명세서는 사용자 경험과 제품 기능 범위를 중심으로 작성한다.
