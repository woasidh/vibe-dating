## 추가 논의: 회원가입 / 로그인

### 질문

- 사용자가 회원 가입을 할 때 성향 태그를 고르고, 이후 언제든 들어가서 데이트 플레이스로 추천받는 방식이면 MVP 회원가입/로그인 기능이 들어가야 하는가?

### 사용자 답변

- 사용자가 회원가입할 때 성향 태그를 고르는 방식이다.
- 이후 언제든 들어가서 데이트 플레이스로 추천받을 수 있는 구조를 생각하고 있다.

### 정리된 결정

회원가입과 로그인은 MVP에 포함한다.

회원가입의 목적은 단순 계정 생성이 아니라 사용자의 기본 데이트 성향 프로필을 저장하는 것이다.
