# 13. 벤치마킹

작성일: 2026-06-14

## 목적

Vibe Dating의 MVP가 어떤 서비스를 대체하려는지, 어떤 서비스와는 정면 경쟁하지 않아야 하는지, 어떤 경험을 차별점으로 가져가야 하는지 정리한다.

이 문서는 기능 명세서의 결정을 보강하기 위한 참고 문서다. 공개 웹/앱스토어 소개를 기준으로 조사했으며, 실제 앱 설치 후 세부 플로우 검증은 별도 UX 리서치에서 다룬다.

## 벤치마킹 기준

- 사용자가 데이트 장소를 찾을 때 실제로 대체재로 쓸 수 있는가
- 장소 탐색, 저장, 비교, 추천, 코스 구성 중 어디에 강점이 있는가
- Vibe Dating이 MVP에서 따라갈 부분과 피해야 할 부분은 무엇인가
- 사용자가 직접 검색하고 비교하는 수고를 줄이는 데 얼마나 기여하는가
- 커플 성향, 오늘의 무드, 자연어 수정 같은 데이트 맥락을 얼마나 잘 다루는가

## 벤치마킹 대상

### 네이버 지도

네이버 지도는 한국 사용자의 장소 검색, 상세 확인, 저장, 예약, 길찾기까지 이어지는 기본 인프라에 가깝다.

공개 소개 기준으로 네이버 지도는 장소/버스/지하철/주소 통합 검색, 발견 탭, 실시간 인기 장소 랭킹, 추천 피드, 핫플레이스, 예약, 내비게이션, 저장, 리뷰, MY 기록 등을 제공한다.

#### 강점

- 한국 로컬 플레이스 데이터와 사용자 습관이 강하다.
- 장소 상세 정보, 리뷰, 저장, 예약, 길찾기 같은 후속 행동이 잘 연결된다.
- 사용자는 최종 확인 단계에서 네이버 지도 링크를 신뢰할 가능성이 높다.

#### 한계 또는 기회

- 사용자가 직접 검색어를 바꾸고, 장소 페이지를 열고, 리뷰를 읽고, 비교해야 한다.
- 장소 추천은 풍부하지만 커플 성향이나 오늘의 데이트 무드에 맞춰 후보를 압축하는 경험은 약하다.
- 좋은 장소를 찾기 전까지 탐색량이 많아질 수 있다.

#### Vibe Dating에 주는 시사점

- 네이버 지도를 대체하려 하지 않는다.
- MVP에서는 외부 상세 확인 링크의 1순위를 네이버 지도로 둔다.
- Vibe Dating은 네이버 지도에 들어가기 전, 사용자가 볼 후보를 줄여주는 큐레이션 레이어로 포지셔닝한다.

### 카카오맵

카카오맵은 빠른 길찾기, 통합 검색, 주변 추천, 지도 위 영역 재검색, 방문자 빅데이터 기반 장소 정보, 즐겨찾기 그룹 공유 등을 제공하는 범용 지도 서비스다.

#### 강점

- 지도 이동 후 해당 영역을 다시 검색하는 흐름이 자연스럽다.
- 맛집, 검색어, 스팟, 축제 같은 주변 추천을 제공한다.
- 즐겨찾기 그룹, 로드뷰, 3D 지도, 대중교통/내비게이션 등 장소 방문 전후 기능이 강하다.

#### 한계 또는 기회

- 데이트 목적의 성향 기반 추천보다는 범용 장소 탐색과 이동 편의에 초점이 있다.
- 사용자가 원하는 분위기, 커플 취향, 코스 맥락을 자연어로 수정하는 경험은 핵심 경험이 아니다.

#### Vibe Dating에 주는 시사점

- 지도 viewport 기반 조회는 사용자가 이미 익숙한 패턴이다.
- Vibe Dating은 현재 지도 영역 안의 후보를 가져오되, 정렬 기준을 검색 랭킹이 아니라 사용자 성향 기반 추천 점수로 둔다.
- 카카오맵도 외부 상세 확인 링크 후보로 유지할 수 있다.

### Google Maps

Google Maps는 전 세계 장소, 리뷰, 사진, 길찾기, 저장 리스트를 제공하는 글로벌 지도 서비스다. Google Play 소개 기준으로 2억 5천만 개 이상의 비즈니스와 장소 정보를 제공하며, 사진/리뷰/유용한 정보를 함께 보여준다.

2026년 3월 Google은 Gemini 기반 Ask Maps를 공개하며 복잡한 자연어 질문에 답하고 개인화된 추천과 지도 결과를 함께 제공하는 방향을 제시했다.

#### 강점

- 글로벌 장소 데이터, 리뷰, 사진, 리스트, 길찾기 연결이 강하다.
- 자연어 기반 지도 탐색이 대형 지도 서비스의 방향으로 부상하고 있다.
- 사용자의 저장/검색 기록을 반영한 개인화 추천으로 발전하고 있다.

#### 한계 또는 기회

- 한국 데이트 장소 탐색의 최종 확인/예약/리뷰 습관은 네이버 지도와 카카오맵 의존도가 높을 수 있다.
- 글로벌 범용 지도 서비스이므로 한국 커플의 데이트 맥락, 상권별 핫플 감도, 장소별 바이브 점수는 별도 도메인 레이어가 필요하다.

#### Vibe Dating에 주는 시사점

- 자연어 기반 장소 탐색은 장기적으로 핵심 경쟁 축이 된다.
- Vibe Dating의 바이브 수정은 단순 챗봇이 아니라 지도, 추천 점수, 플레이스 리스트, 코스 조합을 실제로 바꾸는 편집 경험이어야 한다.
- Google Maps의 방향은 위협이지만, MVP에서는 한국 데이트 맥락에 좁고 깊게 집중하는 방식으로 차별화한다.

### 데이트팝

데이트팝은 2030 놀거리/데이트 콘텐츠 큐레이션 앱이다. App Store 소개 기준으로 직접 체험 후 작성한 전국 5000개 이상 가볼 만한 곳, 2800개 이상 제휴 맛집/놀거리, 큐레이션 콘텐츠, 할인/구독형 상품을 제공한다.

#### 강점

- 데이트/놀거리라는 문제를 직접 겨냥한다.
- 콘텐츠 큐레이션, 방문 후기, 기획전, 할인 상품이 강하다.
- 사용자가 "이번 주 뭐 하지"를 찾을 때 즉시 참고하기 좋다.

#### 한계 또는 기회

- 콘텐츠와 딜 중심 성격이 강해질수록 사용자의 현재 지도 영역, 개인 성향, 즉석 수정 요구를 섬세하게 반영하기 어렵다.
- 사용자는 여전히 마음에 드는 콘텐츠를 고른 뒤 위치, 동선, 상세 확인을 따로 판단해야 할 수 있다.

#### Vibe Dating에 주는 시사점

- 데이트 특화 큐레이션 수요는 이미 검증된 문제다.
- MVP에서 할인, 제휴, 구독, 커머스 기능을 따라갈 필요는 없다.
- 차별점은 "콘텐츠 탐색"보다 "현재 지역 안에서 내 성향에 맞는 후보를 점수화하고 바로 바꾸는 경험"에 둔다.

### 데이트립

데이트립은 로컬 여행과 감각적인 공간 큐레이션에 강한 서비스다. 공식 소개에서는 신뢰하기 어려운 정보 속에서 취향에 맞는 곳을 찾기 어렵다는 문제를 제기하고, 숨겨진 명소와 큐레이션된 로컬 가이드를 제공한다고 설명한다.

#### 강점

- 감각적인 공간, 로컬 큐레이션, 트렌드 발견에 강하다.
- 여행, 맛집, 카페, 문화생활 등 2030 사용자가 선호하는 공간 탐색 맥락과 잘 맞는다.
- 공간 자체의 발견 경험과 콘텐츠 품질이 차별점이다.

#### 한계 또는 기회

- 데이트 전날 특정 상권 안에서 빠르게 후보를 좁히고 코스를 바꾸는 도구라기보다는, 좋은 공간을 발견하고 저장하는 콘텐츠/로컬 큐레이션 성격이 강하다.
- 커플 성향, 오늘 조건, 이동 거리, 카테고리 교체 같은 데이트 계획 편집 맥락은 별도 기회가 있다.

#### Vibe Dating에 주는 시사점

- Vibe Dating도 공간 감도와 큐레이션 품질을 가져가야 한다.
- 다만 MVP의 중심은 콘텐츠 피드가 아니라 지도 기반 추천 결과와 바이브 수정이어야 한다.
- 장소 요약과 추천 이유는 감각적이되, 확인되지 않은 외부 문장을 재사용하지 않고 자체 문장으로 작성한다.

### 완벽한 하루

완벽한 하루는 데이트코스/관광코스 맞춤형 추천 앱이다. App Store 소개 기준으로 SNS와 블로그 탐색, 광고성 정보, 지도 확인, 반복 비교의 번거로움을 문제로 제기하고, 사용자 성향 기반 맞춤형 장소 추천과 코스 설계를 제공한다.

#### 강점

- Vibe Dating이 해결하려는 문제와 가장 직접적으로 겹친다.
- 사용자 성향, 맞춤형 장소 추천, 코스 설계, 예상 소요시간/지출금액 같은 계획 요소를 다룬다.
- 사용자가 여러 채널을 오가며 비교하는 수고를 줄인다는 메시지가 명확하다.

#### 한계 또는 기회

- 공개 소개만으로는 지도 viewport 기반 추천, 성향별 상대 점수, 자연어 기반 추천 수정이 얼마나 핵심 경험인지 확인하기 어렵다.
- 코스 설계 중심으로 보일 경우, Vibe Dating은 먼저 장소 후보를 잘 좁히고 사용자가 요청할 때 코스를 만드는 방식으로 더 가볍게 시작할 수 있다.

#### Vibe Dating에 주는 시사점

- 문제 정의는 이미 시장에서 공감되는 영역이다.
- MVP 차별점은 "사용자 성향 기반 지도 추천"과 "바이브 수정으로 추천 결과/코스를 즉시 편집"하는 경험으로 더 선명하게 잡아야 한다.
- 완성된 코스 추천보다, 사용자가 마음에 들 때까지 바꿀 수 있는 추천 편집 경험을 핵심으로 둔다.

## 대체 행동 벤치마킹

### 네이버 지도 직접 탐색

사용자는 지역명과 카테고리를 검색하고, 플레이스 페이지를 여러 개 열어보고, 리뷰와 사진을 비교한 뒤 후보를 고른다.

이 방식은 정보량은 많지만, 사용자가 직접 후보를 줄여야 한다. Vibe Dating이 줄여야 하는 핵심 수고가 여기에 있다.

### 블로그, 인스타그램, 숏폼 탐색

사용자는 감도 있는 장소와 최신 핫플을 발견하기 위해 블로그, 인스타그램, 숏폼 콘텐츠를 본다.

이 방식은 영감 발견에는 강하지만 광고성 콘텐츠, 저장/비교의 어려움, 위치/동선 확인의 번거로움이 있다.

### 일반 LLM에게 데이트 코스 질문

사용자는 ChatGPT 같은 일반 LLM에게 지역과 취향을 말하고 코스를 추천받을 수 있다.

이 방식은 자연어 대화에는 강하지만, 특정 시점의 운영 여부, 지도 영역, 외부 링크, 실제 플레이스 데이터, 추천 결과의 지도 시각화가 약할 수 있다. Vibe Dating은 LLM을 단독 답변자가 아니라 자체 Place DB와 지도 UI를 조작하는 추천/수정 엔진으로 써야 한다.

## MVP에 반영할 결정

- Vibe Dating은 지도 서비스를 대체하지 않고, 지도 서비스 위의 데이트 큐레이션 레이어로 포지셔닝한다.
- MVP의 첫 화면은 콘텐츠 피드나 랜딩 페이지가 아니라 지도 기반 추천 화면이어야 한다.
- 추천 결과는 현재 지도 영역 안의 플레이스를 사용자 성향 기반 추천 점수순으로 보여준다.
- 플레이스 카드에는 장소 상세를 길게 복제하지 않고, 간단 요약, 추천 이유, 평점, 외부 링크를 제공한다.
- 외부 상세 확인은 네이버 지도 링크를 1순위로 두고, 카카오맵/Google Maps 링크를 보조로 검토한다.
- 데이트팝처럼 딜/커머스/구독을 MVP에 넣지 않는다.
- 데이트립처럼 감각적인 공간 발견은 참고하되, MVP의 중심은 콘텐츠 소비가 아니라 추천 결과 편집이다.
- 완벽한 하루와 겹치는 "맞춤형 데이트 추천" 메시지는 피할 수 없으므로, 지도 viewport 기반 추천과 자연어 바이브 수정으로 차별점을 선명하게 만든다.
- 일반 LLM보다 좋은 경험을 만들려면, 추천 답변이 지도 마커, 리스트 정렬, 코스 조합으로 실제 반영되어야 한다.

## 제품 차별화 문장

Vibe Dating은 데이트 장소 콘텐츠 앱이 아니라, 현재 지도 영역 안에서 내 데이트 성향에 맞는 장소를 먼저 좁혀주고, 자연어로 추천 결과와 코스를 바꿀 수 있게 해주는 지도 기반 데이트 큐레이션 서비스다.

## 후속 확인 필요

- 주요 경쟁 앱을 실제 설치해 온보딩, 추천, 지도, 저장, 코스 생성, 결제/딜 플로우를 캡처한다.
- 서울 초기 오픈 후보 상권에서 네이버 지도, 카카오맵, 데이트팝, 데이트립의 추천 결과를 같은 조건으로 비교한다.
- 성향 문장별로 기존 서비스가 제공하지 못하는 추천 기준을 정리한다.
- 자연어 수정 요청이 실제로 어떤 UI 변화를 만들어야 사용자가 차별점을 느끼는지 프로토타입으로 검증한다.
